🚀 Setup Instructions:

  1. Create a Google Drive folder for your Duke Chapel analytics data
  2. Create separate Google Sheets for each data source:
    • GA_Demographics - Google Analytics demographic data
    • GA_Traffic_Acquisition - Traffic sources and channels
    • GA_Pages_And_Screens - Page performance data
    • GA_UTMs - Campaign tracking data
    • FB_Metrics - Facebook metrics (with tabs: Reach, Interactions, Follows, Views, Link_Clicks, Visits)
    • FB_Audience - Facebook audience demographics
    • IG_Metrics - Instagram metrics (with tabs: Reach, Interactions, Follows, Views, Link_Clicks, Visits)
    • IG_Audience - Instagram audience demographics
    • Email_Campaign_Performance - Email marketing data
    • YouTube_Content - YouTube video performance
    • YouTube_Demographics - YouTube audience (with tabs: Age, Gender, Geography, Subscription_Status)
  3. Make the folder publicly viewable (Share → Anyone with the link can view)
  4. Get a Google Drive API key from Google Cloud Console (enable both Drive and Sheets APIs)
  5. Copy the Folder ID from the Drive URL (after /folders/)
  6. Enter both values above and click Connect

Pro Tip: Use consistent naming! The sheet names must match exactly as listed above.
